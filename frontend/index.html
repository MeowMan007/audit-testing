⚠️ Important Limitations

Understanding WCAG Accessibility

The Web Content Accessibility Guidelines (WCAG) 2.1 define how to make web content more accessible to people with disabilities. Our tool checks for these key principles:

🎨

Color Contrast

Text must have sufficient contrast against its background. WCAG requires at least 4.5:1 for normal text and 3:1 for large text (18px+). Low contrast makes content unreadable for users with low vision.

WCAG 1.4.3 — Level AA
🖼️

Alt Text

Every informational image must have a descriptive alt attribute. Screen readers read this text aloud to users who cannot see the image. Decorative images should use alt="".

WCAG 1.1.1 — Level A
📐

Semantic HTML

Proper use of HTML elements (<nav>, <main>, <h1>-<h6>) creates a meaningful document structure that assistive technologies rely on for navigation.

WCAG 1.3.1 — Level A
📊

Accessibility Score

Our scoring system starts at 100 and deducts points for each violation found. Critical issues (like missing alt text) deduct more than warnings (like generic link text). The CNN model adds AI-detected deductions.

Grades: A (90+) → F (below 60)

Known Limitations

🤖

AI Model Limitations

The EfficientNet-B0 CNN model is trained on a synthetic dataset of only 500 samples. It may miss real-world edge cases, produce false positives on unusual layouts, and cannot fully replace human expert review.

📋

Rule-Based Coverage

Automated rule checks cover approximately 30-40% of WCAG 2.1 criteria. Many accessibility issues (like keyboard trap detection, cognitive load, or screen reader compatibility) require manual testing.

📦

Small Dataset

The training dataset contains 500 synthetic samples — far fewer than production ML systems. Model accuracy improves with larger, more diverse real-world datasets (ideally 10,000+ labeled screenshots).

⚡

Dynamic Content

Single-page applications (SPAs), lazy-loaded content, and JavaScript-heavy sites may not be fully captured. The tool takes a snapshot at load time and may miss content that appears after user interaction.
